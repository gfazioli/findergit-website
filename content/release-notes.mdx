# Release Notes

Below is an overview of the latest changes and improvements made to FinderGit.

## v1.0.0 — April 2026

### Initial Release

FinderGit launches as a native macOS Git-aware file browser.

**Features:**
- Expandable tree view with sortable columns (Name, Branch, Status, Changes, Size, Date)
- Live Git status for all repositories via FSEvents
- Inline diff viewer with colored additions/deletions
- Git actions: stage, unstage, commit, push, pull, fetch
- Smart context menus adapting to file type and Git state
- Quick Look support (Space bar)
- Search and filter with "Git Only" toggle
- Multiple root folders with drag-and-drop sidebar
- Sub-directory dirty indicators
- Settings: scan depth, font size, excluded folders, terminal app, Git path
- Full menu bar with keyboard shortcuts
- Custom app icon

**Requirements:**
- macOS 15 (Sequoia) or later
- Git (bundled with macOS)
