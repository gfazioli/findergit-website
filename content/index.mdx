# FinderGit

**FinderGit** is a native macOS application that combines file browsing with Git intelligence. Instead of switching between Finder and a Git client, you get everything in one window.

![FinderGit main window](/screenshot-hero.png)

## What it does

- **Tree view with sortable columns** вҖ” browse your files like Finder's list view, with columns for Branch, Status, Changes, Size, and Date Modified
- **Live Git status** вҖ” every repository shows its current branch, clean/dirty/unpushed state, updated in real time via FSEvents
- **Ahead/behind counter** вҖ” the status badge shows `вҶ‘N`, `вҶ“N`, or `вҶ‘N вҶ“M` with tooltips so you can spot repos that need a push, a pull, or both without opening them
- **Auto-fetch** вҖ” optional background `git fetch` at a user-chosen interval so the ahead/behind counter stays fresh without manual refreshes
- **Inline diff viewer** вҖ” click any modified file to see a colored diff with line numbers
- **Git actions** вҖ” stage, unstage, commit, push, pull, fetch, and switch branches directly from the app
- **Smart context menus** вҖ” right-click adapts to context: file operations for regular files, Git operations for repositories
- **Quick Look** вҖ” press Space to preview any file. `.md` and `.markdown` files open in a native renderer (GitHub-style theme); everything else falls back to system Quick Look
- **Search & filter** вҖ” filter the tree by name or toggle "Git Only" to show just repositories
- **Multiple root folders** вҖ” add as many root directories as you want; drag folders from the macOS Finder into the sidebar to add them as roots
- **Universal binary** вҖ” one DMG runs on Apple Silicon and Intel Macs; macOS picks the right slice at launch
- **Auto-updates** вҖ” built in via Sparkle, signed with EdDSA

## Requirements

- macOS 15 (Sequoia) or later
- `git` вҖ” provided by the Xcode Command Line Tools. On a fresh Mac, run `xcode-select --install` and accept the license with `sudo xcodebuild -license` before launching FinderGit (see [Getting Started](/docs/getting-started))

## Architecture

FinderGit is built with **SwiftUI** and the **Observation** framework. The core Git logic lives in a reusable `FinderGitCore` framework that wraps the `git` CLI with async/await APIs.
