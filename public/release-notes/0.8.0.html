FinderGit 0.8.0

Release notes

In-app file preview ✨

FinderGit can now preview files inside the window — no more bouncing out to a separate panel.

  • Code with syntax highlighting — press Space (or ⌘Y) on any source file and read it in colour, across dozens of languages, themed to match light and dark mode.
  • Rendered Markdown — .md files render formatted, on a clean uniform background.
  • Images, PDFs & media — open in a native preview right in the window, and gracefully fall back when a file type can't be rendered.
  • Finder-style toggle — Space opens and closes the preview, Esc dismisses it, just like the Finder.

Support FinderGit 💛

A new Sponsor entry in the Help menu and the About tab, for anyone who'd like to back FinderGit's development.

Refinements

  • The What's New panel now appears on every update, patch releases included.
  • Stability fixes in the preview pipeline.
