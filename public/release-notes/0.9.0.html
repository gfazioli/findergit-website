FinderGit 0.9.0

Release notes

Reclaim your disk space 🧹

FinderGit 0.9.0 adds a Maintenance tab to every repository:

  • Disk usage report — see exactly where the Git data goes: packs, loose objects, undo history, with an honest estimate of what a cleanup can give back.
  • Optimize — one-click safe housekeeping. Your undo history stays intact; run it anytime.
  • Deep Clean — maximum reclaim: clears the undo history and prunes unreachable data immediately. Your stashes are kept by default, and every knob — prune horizon, aggressive repack — is adjustable.
  • Size column — every repo's Git footprint at a glance in the file browser. Sortable, and it refreshes itself after a cleanup.

Settings, redesigned ⚙️

A cleaner Settings window: grouped options with a short explanation under every control, in a regular resizable window.

Refinements

  • The in-app preview pipeline got several stability fixes.
