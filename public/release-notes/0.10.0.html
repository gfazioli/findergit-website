🛡 Repo Trust — know what a repo runs before you trust it

Cloning or pulling a repository can quietly arm it to run code — the moment you open the folder, the first time an AI agent loads it, or as soon as you install dependencies. Repo Trust surfaces that auto-run surface for every repo so you can review it first.

  • Auto-run surface, in plain language — a new Trust tab lists every place a repo can run code (editor folder-open tasks, AI-agent configs, install/test scripts), with a plain-language "what runs" and a full-file preview. It only ever reads — nothing is executed.
  • A badge across the browser — a yellow shield marks repos with auto-run hooks you haven't trusted; trust one and it clears.
  • Changed since last seen — the high-signal alert: when a repo's auto-run surface changes after a pull, an orange badge flags it — even on repos you'd trusted — because that's exactly when a trusted dependency can turn hostile.

✨ A new About & Support window

  • A redesigned About window (also shown at launch — you can turn that off in Settings) with a clean way to support development and discover the developer's other apps.
