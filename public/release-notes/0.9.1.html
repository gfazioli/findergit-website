FinderGit 0.9.1

Release notes

Hardening 🔒

A safety pass on Repo Maintenance for repositories that come from untrusted sources:

  • Maintenance now acts only on a repository's real Git directory and won't follow a crafted pointer to somewhere else on disk.
  • Reflog cleanup ignores anything that isn't a genuine Git reference.
  • Pushing rejects malformed branch names.

No behavior change for normal repositories — Optimize, Deep Clean and the Size column work exactly as before.
