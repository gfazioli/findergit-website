Release notes

FinderGit 0.6.1 — small touches

A maintenance release with one user-visible polish and one under-the-hood improvement.

What's new

  • The "Made with ❤️ by Undolog" credit in the About panel now opens the Undolog studio page. Previously it sent you to the plain GitHub profile — the new destination matches what the link actually says.

Under the hood

  • The auto-update internals are now ready for Swift's strict concurrency model. No user-visible behavior change today; the groundwork lands now so the next Swift language-mode migration is a no-op for FinderGit.

Compatibility

  • macOS 15 (Sequoia) or newer.
  • Universal binary: native on Apple Silicon and Intel.
  • Signed with Apple Developer ID and notarized by Apple.

Auto-update users on 0.6.0 will receive this build the next time FinderGit checks; manual download links are below.
